ArtEverywhere helps the user discover various forms of public art near them, specifically in New York City.


Categories :iOS, UI/UX Design, Graphic Design

Released :3 Nov 2015

See on Github

At the Flatiron School, we were encouraged to work with a partner, create a project, and present it to a weekly meetup hosted by the Flatiron School. While working with Joel, we brainstormed ideas, participated in paired-programming and collaborated in designing the user interface. We went through several code reviews with instructors to create a bug-free and efficient application.


In the creation of this app, I contributed with the following:
  • implemented MapKit and CoreLocation frameworks to provide relevant location information
  • utilized UIScrollView class and UITapGesture REcognizer to deliver image assets and text content to user
  • designed graphics and user interface to provide a cohesive user experience

features

  • discover public art near you
  • map that displays the location of the art
  • lightbox gallery for photos
  • description for each artwork

technologies

  • Objective-C | Xcode
  • Adobe Illustrator
