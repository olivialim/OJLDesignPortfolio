Novel is an iOS application that brings users all the benefits of Project Gutenberg and Goodreads into one simple application.

Novel was designed to allow users to search eBooks from Project Gutenbergs’ database, download eBooks from Project Gutenberg, and connect with other users through Facebook. Novel also implements Goodreads API, allowing users to see reviews for books found in Project Gutenbergs’ library.

All downloaded books will be exported and opened in iBooks.


Categories :iOS Development, UI/UX Design

Released :30 Dec 2015

See on Github

Working with in a group of four at the Flatiron School, I was able to brainstorm and ideate, while creating efficient and powerful code. For three weeks, we went through daily code reviews with instructors and incorporated various technologies, for both learning and experience sake.


In the creation of this app, I contributed with the following:
  • designed branding, graphics, and layouts for every screen seen by user
  • parsed through Project Gutenburgs' XML data
  • synthesized Goodreads API to pull book reviews and descriptions for accurate information for every book
  • incorporated Parse to store user data and create chat rooms
